PROJEK

Dashboard Untuk Security (cetak surat izin masuk)

Proyek dashboard ini dibuat menggunakan Google Looker Studio untuk menampilkan data rekapitulasi surat izin masuk secara interaktif. Security bisa mengetahui data secara real-time dan security bisa langsung melakukan cetak suat izin masuk untuk tamu.

Slicer Pada Excel

Proyek rekap logistik ini dilengkapi dengan fitur slicer untuk mempermudah pencarian dan analisis data. Dengan slicer, pengguna dapat langsung memfilter informasi berdasarkan tanggal tertentu. Pencarian data muatan menjadi lebih cepat karena hanya menampilkan jenis barang yang dipilih.
